ĐỀ TÀI: TÌM HIỂU VỀ MICROSOFT AZURE

Sinh viên: Nguyễn Vũ Ngọc Linh - 22DH111959

Lớp: CNTT - HUFLIT

Giới thiệu

Microsoft Azure là một nền tảng điện toán đám mây được phát triển bởi Microsoft, cung cấp hơn 200 dịch vụ cho phép triển khai, quản lý và xây dựng ứng dụng trên quy mô toàn cầu.

Mục tiêu đề tài

Nghiên cứu tổng quan về Microsoft Azure, cách tạo Web App, triển khai ứng dụng từ GitHub và khai thác các dịch vụ như App Services, Storage, Database.

Kết luận

Azure giúp đơn giản hóa việc phát triển và triển khai ứng dụng, là nền tảng quan trọng trong xu hướng điện toán đám mây hiện nay.
